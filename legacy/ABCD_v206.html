Overview

This is ABCD version 2.06. For more information on ABCD in general please see the general ABCD resource directory. Several minor updates have been done to ABCD2.06, the ratified TDWG standard, that are backwards compatible and do not require a change in the namespace. For more details see Updates to ABCD2.06.

Note: While this document is human-readable, RDDL-aware processors may use it to automatically retrieve resources such as different schema versions for validating documents. See http://www.rddl.org for further information.
