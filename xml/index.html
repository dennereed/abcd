Overview

Here you can find the ABCD XML Schemata. Several minor updates have been done to ABCD 2.06, the ratified TDWG standard, that are backwards compatible and do not require a change in the namespace. Note: While this document is human-readable, RDDL-aware processors may use it to automatically retrieve resources such as different schema versions for validating documents. See http://www.rddl.org for further information.

Notes

Although RDDL-aware processors may automatically detect a schema location, most processors require an explicit location. ABCD instance documents intended for validation should therefore contain the following attributes in the "Datasets" root element:

xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:schemaLocation="http://www.tdwg.org/schemas/abcd/2.06 http://rs.tdwg.org/abcd/2.06/ABCD_2.06.xsd"
